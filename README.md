### About

Final-year Software Engineering student (ISIMG, Tunisia) building IoT + ML systems for elder care. My graduation project is a real-time **fall detection wearable** — ESP32 + IMU → XGBoost model (95.45% recall, AUC 0.9976) → live alerts to caregivers. I've since extended that work into unsupervised **gait anomaly detection**, benchmarking Isolation Forest, LSTM Autoencoders, and SHAP explainability on clinically-motivated gait perturbations.

- 🔭 Currently working on turning fall/gait detection research into a broader health-tech IoT/ML platform
- 🎓 Applying for master's programs + the MEXT scholarship for 2026/2027
- 🌱 Learning Japanese and leveling up my ML systems knowledge
- 💬 Ask me about embedded ML, time-series anomaly detection, or wearable sensor pipelines
